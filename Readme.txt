Przed wykonaniem poniższej instrukcji, należy mieć wcześniej zainstalowany node.js na swoim komputerze
Aby korzystac z projektu należy lokalnie w folderze lokalnie wykonać poniższe kroki
1.Otworzyć gitbash w folderze a nastepnie wpisac

git init
git remote add origin https://github.com/JanuszProgramowaniaa/Anielska_Kuchnia.git
git pull origin master

2.Po otworzeniu folderu z aplikacją odpalic komendy 
npm install

3.Aby uruchomić aplikację
npm run start
